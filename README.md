# my-discord-bot

サークル運営用の Discord ボットを試作したリポジトリです。アンケート用の導線と、簡単な定型応答コマンドをまとめています。

## 概要

- `hello` メッセージへの自動返信
- `/ping` などのスラッシュコマンド
- 曜日ごとの日程調整メッセージ送信

## 実行

```bash
python main.py
```

## 必要な設定

- Discord Bot のトークンを `TOKEN` に設定
- `discord.py` 系の依存関係をインストール

## コマンド例

- `/ping`
- `/greeting`
- `/mon` 〜 `/sun`

## 実装メモ

- `discord.Bot` を使ったイベント駆動構成
- メッセージイベントとアプリケーションコマンドを併用
